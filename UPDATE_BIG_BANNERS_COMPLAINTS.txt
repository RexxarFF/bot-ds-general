FUNFERNUS UNIFIED BOT 1.1
БОЛЬШИЕ БАННЕРЫ + ЖАЛОБЫ

ЧТО ИЗМЕНЕНО
1. Панель «Подать заявку» теперь Components V2:
   - большой баннер сверху;
   - заголовок, текст, подпись и кнопка в одной рамке.

2. Панель технической поддержки теперь Components V2:
   - большой баннер сверху;
   - все кнопки в одной рамке;
   - добавлена кнопка «Жалоба».

3. Жалобы:
   - «На игрока»;
   - «На другое»;
   - отдельные номера REPORT-0001;
   - отправка в канал рассмотрения поддержки;
   - сотрудники могут ответить и закрыть обращение.

4. В настройках сообщества появилась кнопка «Текст панели».

КАК ОБНОВИТЬ
1. Сохраните старый файл .env.
2. Замените файлы бота новым архивом.
3. Верните .env.
4. Установите зависимости: pip install -r requirements.txt
5. Запустите main.py.
6. Выполните /настроить_бота.

КАК ЗАГРУЗИТЬ БАННЕР ЗАЯВКИ
/настроить_бота → Заявки → Баннер → Загрузить / заменить → Опубликовать.

КАК ЗАГРУЗИТЬ БАННЕР ТЕХПОДДЕРЖКИ
/настроить_бота → Сообщество → Баннер панели → Опубликовать панель.
