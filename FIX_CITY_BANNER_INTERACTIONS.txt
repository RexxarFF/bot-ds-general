ИСПРАВЛЕНИЕ ОШИБОК ПАНЕЛИ ГОРОДОВ — 23.07.2026

Исправлено:
1. TypeError / AttributeError при view=None
   discord.py 2.7 не принимает явное значение view=None в response.send_message()
   и followup.send(). Теперь параметр view полностью исключается из запроса,
   когда у сообщения нет компонентов.

2. Бесконечный индикатор «бот думает»
   После defer(thinking=True) результат установки баннера завершает исходный
   ответ через interaction.edit_original_response(). При резервной отправке
   локальный файл создаётся заново, чтобы не использовать уже закрытый File.

3. Лишние PATCH-запросы и Discord 429
   При смене баннера заявки обновляется только панель заявок, а при смене
   баннера управления — только панель управления. Раньше изменялись сразу обе.
   Публикация панелей одного сервера сериализована через asyncio.Lock.
   При временной HTTP-ошибке бот больше не создаёт дубликат панели.

4. Ошибки компонентов панели настройки
   Временные Select-меню и Modal баннера получили обработчики on_error:
   ошибка записывается в лог, а пользователь получает понятное уведомление.

Изменён только modules/cities.py. Системы бизнеса, банка, тикетов, заявок,
правительства и другие модули не переписывались.
