ИСПРАВЛЕНИЕ ПУБЛИКАЦИИ ПАНЕЛЕЙ ГОРОДОВ

Причина ошибки 50035:
Старое сообщение панели имело необратимый Discord-флаг IS_COMPONENTS_V2.
Discord запрещает использовать embeds у такого сообщения, даже если новая версия
бота уже использует обычный discord.ui.View.

Что изменено:
1. Перед редактированием бот проверяет бит IS_COMPONENTS_V2 (1 << 15).
2. Несовместимая старая панель больше не редактируется через PATCH.
3. Бот сначала создаёт новую панель с обычным Embed, большим локальным баннером
   и legacy View-кнопкой.
4. После успешной публикации старая Components V2 панель удаляется.
5. Новый ID панели сохраняется в общем JSON-хранилище.
6. Добавлена резервная обработка Discord 50035, если конфликт обнаружился только
   при ответе API.

Действие после обновления:
Перезапустить бота и один раз нажать «Опубликовать панели» либо повторно установить
баннер панели. Миграция выполнится автоматически.
